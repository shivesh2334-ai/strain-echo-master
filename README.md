# StrainEcho Master 🫀

[![Deploy with Vercel](https://vercel.com/button)](https://vercel.com/new/clone?repository-url=https://github.com/yourusername/strain-echo-master)

A comprehensive, interactive learning platform for **Clinical Applications of Strain Echocardiography (2026)**.

## Features

- 📚 **4 Major Categories**: Technology Updates, Normal Ranges, Clinical Applications, Future Directions
- 🎓 **Expert-Curated Content**: Resources from Stanford, Mayo Clinic, Johns Hopkins, MIT, ASE
- ✅ **Progress Tracking**: Local storage saves your learning progress
- 🎥 **Video Integration**: Direct links to high-quality educational videos
- 💡 **Clinical Tips**: Actionable learning tips for each parameter
- 🔍 **Smart Search**: Find topics, tips, and videos instantly
- 📱 **Responsive Design**: Works on desktop, tablet, and mobile

## Educational Resources Included

### Free Video Content:
- **ASE360 YouTube Channel**: Official American Society of Echocardiography content [^1^][^3^][^9^]
- **Mayo Clinic 2024**: Applying Strain in Practice – Chemotherapy and Beyond [^22^]
- **Cleveland Clinic**: Strain Imaging - Common Errors and Artifacts [^19^]
- **MIT OpenCourseWare**: Machine Learning for Cardiac Imaging [^14^]

### Key Topics Covered:
- Strain Standardization Task Force guidelines
- STE vs TDI technical comparison
- Normal GLS ranges (> -18%)
- Cardio-oncology monitoring protocols
- Heart failure assessment (HFrEF/HFpEF)
- Ischemic heart disease applications
- Valvular heart disease evaluation
- HCM and special populations
- 3D Strain and AI integration

## Tech Stack

- React 18 + Vite
- Tailwind CSS
- Lucide React Icons
- Framer Motion (animations)
- LocalStorage API (progress tracking)
